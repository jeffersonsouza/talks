Segurança

Dicas

Itens básicos para ajudar na segurança Global

Sempre utilize um usuário para a sua aplicação

Você pode definir o usuário passando o parâmetro -u ao criar o seu container.

docker run -dit --name="myapp" -u user:group myapp
Utilize containers em que você confie. Você pode sempre verificar a criação do container a partir do Dockerfile.
Sempre verifique os pacotes instalados, procurando por vulnerabilidades.
Utilize networks para comunicação entre suas aplicações.
Se não precisa expor a porta, não exponha.
Previna-se contra ataques DDoS.

Para mais informações, visite o manifesto: https://benchmarks.cisecurity.org/tools2/docker/CIS_Docker_1.11.0_Benchmark_v1.0.0.pdf
